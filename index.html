Git的基本操作

发表于 2023-03-25 更新于 2023-03-26

Git 一些基本知识

git clone [github上的项目的地址] 白嫖代码

git init 管理某个文件夹下的代码，执行后会有一个.git的文件夹，这个文件夹以外的空间叫工作区

git add . “.”表示为当前文件夹 这句命令是让当前文件夹所有文件变成准备提交的状态

git commit -m “[对这次提交进行备注]” git 可以把源代码保存在仓库中

git log 可以查看提交历史记录

git checkout HEAD [代码文件名] 恢复出问题的代码文件

git add [提交代码文件名] 可以只让某个文件呈准提交状态

add是让文件进入暂存区，暂存区也是git的特色之一
