  1. branch问题

    因为github在教程发布的后面时间里更新了，branch的master都默认变为main，这些也是后面搜了hexo上传github后404问题才知道的。把站点配置文件里的master改成main以及github pages 处也改成main，就基本没事了。

  2. hexo部署时警告

​ git 执行hexo g -d命令时总是出现这些警告，到网上也搜了好久才搜到有一样的情况，出现这个警告后，先执行 hexo clean，清除之前的部署，然后输入hexo config core.autocrlf,结果会出现ture，然后再hexo g和hexo d，虽然还是有警告，但是不知道为什么，反正就成功了(手动狗头)。
